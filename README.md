# markdown-it-prismjs

## Install

```bash
npm install @iktakahiro/markdown-it-prismjs --save
```

## Usage
```typescript
import MarkdownIt = require("markdown-it")
import markdownItPrism from "@iktakahiro/markdown-it-prismjs"

const md = MarkdownIt()
md.use(markdownItPrism)
```
